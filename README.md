# bevy-games

A collection of games built with [Bevy](https://bevyengine.org/) to learn game development and Rust.

## Projects

| Project | Description |
|---------|-------------|
| [bevy-asteroids](bevy-asteroids/) | 3D inertial Asteroids-like game |
| [bevy-space-rts](bevy-space-rts/) | Top-down space RTS with base building and supply chains |

## Prerequisites

- [Rust](https://rustup.rs/) (stable)

## Building

```bash
cargo build              # Build all projects
cargo run -p bevy-asteroids   # Run a specific game
```

## Linting & Tests

```bash
cargo clippy             # Lint all projects
cargo test               # Test all projects
```
