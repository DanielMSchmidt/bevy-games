# Development Workflow

1. Create feature branch:
   dev/<feature>

2. Copy ITERATION_TEMPLATE.md into:
   docs/iterations/<feature>.md

3. Fill out questions before coding.

4. Implement feature.

5. Request AI review.

6. Fix issues.

7. Merge only after explicit Merge approval.

---

## Rule

Never merge without review.
Never skip acceptance criteria.
Never implement more than one feature per branch.
